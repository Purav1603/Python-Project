**#Library Management System:**
This project is written using python code to create a software for managing Members of Library, Staffs of library, and books in the library, This is to track which books are avilable, borrowed, and by whom. 

**#Features:**
- Add new books into the Library.
- To register new members.
- Add data of borrowed books and returned books.
- Load and Save data from Comma seperated value (csv) file.
- Search for books in library using author or by the title of the book.

**#Installation:**
- To run this code, you will need Python 3 Instaalled on your device.
- Download this repository.
- open the folder in IDLE or in the terminal.

**#Usage:**
Run the program with the file main.py.

The Menu will be like:
1.view all Menu 
2.Add a new book
3.search for a book
4.save books to file 
5.Exit

**#Files:**

- main.py - The entry point of the software and the menu
- book.py - This file contains the Book class
- member.py - This file contains the Member class
- librarian.py - This file contains the librarian class
- library.py - This file contains the Library class it handles the the file storage
- books.csv - This file saves the books records
